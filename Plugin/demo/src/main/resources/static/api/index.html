1. 插件接口

1.1. 获取插件信息

URL: http://localhost:8080/plugin/{envId}

Type: GET

Author: AD

Content-Type: application/x-www-form-urlencoded;charset=utf-8

Description: 获取插件信息

Path-parameters:

ParameterValueTypeRequiredDescription

envId

string

true

环境id

Response-fields:

FieldTypeDescriptionSince

envId

string

插件环境id

-

envName

string

插件环境名

-

name

string

插件名(文件名)

-

size

string

jar包大小(单位: B,KB,MB,GB)

-

beans

array

bean数组

-

     └─name

string

bean名称

-

     └─modifyName

boolean

是否修改bean名称

-

     └─srcName

string

bean原始名称

-

     └─classname

string

bean类名

-

     └─scope

string

作用域

-

Response-example:

[
  {
    "envId": "95",
    "envName": "louann.nitzsche",
    "name": "louann.nitzsche",
    "size": "n6gjsc",
    "beans": [
      {
        "name": "louann.nitzsche",
        "modifyName": true,
        "srcName": "louann.nitzsche",
        "classname": "louann.nitzsche",
        "scope": "r31qka"
      }
    ]
  }
]

Curl-example:

curl -X GET -i http://localhost:8080/plugin/linux,mac,window
